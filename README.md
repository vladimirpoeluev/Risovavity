# Risovatity
## Описание
Данная система является сервисом для публикации своих рисунков и ключей подключения к интерактивным холстам
## Как запустить?
Для запуска приложения необходимо установить MS Sql Server. Далее, восстановить базу данных из скрипта предоставленном в исходнике проекта. После восстановления базы данных, необходимо изменить файл конфигурации приложения appConfig.json.
